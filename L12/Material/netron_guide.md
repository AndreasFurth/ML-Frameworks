# Netron (Model Visualization)

## What it does
- Visualizes ONNX, TorchScript, TensorFlow, and other model formats
- Shows layer graph, shapes, and parameters

## Quick usage
1. Export a model (ONNX or TorchScript)
2. Open Netron and load the model file
3. Inspect inputs, outputs, and layer sizes

## Tips
- Use ONNX for cross-framework inspection
- Save a small test model for quick demos
